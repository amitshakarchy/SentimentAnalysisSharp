# SentimentAnalysisSharp
AFINN sentiment analysis in c# : Wordlist-based approach for sentiment analysis.

# Example

- ReddictCommentExample: Sentiment analysis per subreddit (based on comments)

# Features

- En/fr support
- Emoticon support
